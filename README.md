## Featured Projects

### 🏏 Cricket & Capital
> *Does India winning affect the stock market? A data journalism investigation.*

Analyzed 10+ years of Indian cricket match results against Sensex daily movement. Built a fully interactive HTML dashboard revealing correlation patterns between match outcomes and market behavior.

**Tech:** Python · Pandas · JavaScript · Chart.js · HTML/CSS  
**Key finding:** Post-victory trading sessions show measurable upward bias in FMCG and media sectors.

[![View Project](https://img.shields.io/badge/View_Dashboard-00F5D4?style=flat-square&logo=github&logoColor=black)](https://kishandatalab.github.io/cricket-and-capital)

---

### 🗳️ Ghost Voters
> *Statistical anomaly detection in Indian electoral rolls.*

Investigated constituency-level voter registration data for patterns inconsistent with demographic norms — abnormal growth rates, impossible density clusters, and sequential ID irregularities. Interactive investigation dashboard.

**Tech:** Python · Pandas · SQL · JavaScript · D3.js  
**Approach:** Z-score outlier detection + geospatial clustering on public ECI datasets.

[![View Project](https://img.shields.io/badge/View_Dashboard-F72585?style=flat-square&logo=github&logoColor=white)](https://kishandatalab.github.io/ghost-voters)
